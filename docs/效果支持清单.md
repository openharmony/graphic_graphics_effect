## 四、效果创建流程

```
用户创建效果：
GERender::GenerateShaderFilter() 或 GEVisualEffect::GenerateShaderMask()
  ↓
工厂创建：GEEffectFactory::Create()
  ↓
数组索引查找 O(1)
  ↓
返回效果对象
```

所有效果创建已完全由工厂接管，无回退机制。
