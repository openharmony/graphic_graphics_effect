# 效果工厂（GEEffectFactory）设计文档

## 一、概述

### 1.1 什么是效果工厂？

效果工厂（`GEEffectFactory`）是 Graphics Effect（GE）组件的核心基础设施，提供了统一的效果创建和管理机制。通过工厂模式，实现了：

- **集中注册**：所有效果类型在一个文件中集中注册，- **O(1)查找**：通过哈希表快速查找和创建效果
- **统一接口**：提供统一的效果创建接口
- **回退机制**：保留原有的 switch-case 作为安全保障

### 1.2 设计目标

1. **性能优化**：从 O(n) 的 switch-case 查找优化为 O(1) 的哈希表查找
2. **代码维护**：新增效果只需添加一行注册代码
3. **扩展性**：支持外部动态加载效果
4. **兼容性**：完全兼容现有的参数系统

### 1.3 适用场景

- 需要频繁创建效果的渲染管线
- 支持动态加载外部效果库
- 需要统一管理大量效果类型
- 需要快速查找和创建特定效果

### 2.1 工厂模式架构

```

│                    GEEffectFactory                            │
│  ┌─────────────────────────────────────────────────────┐   │
│  │              Singleton Instance                       │   │
│  │  - GetInstance()                                     │   │
│  │  - CreateFilter/CreateMask/CreateShape               │   │
│  └─────────────────────────────────────────────────────┘   │
│  ┌─────────────────────────────────────────────────────┐   │
│  │              Registration Table                       │   │
│  │  - std::unordered_map<FilterType, CreatorFunc>      │   │
│  │  - RegisterFilter/RegisterMask/RegisterShape        │   │
│  └─────────────────────────────────────────────────────┘   │
│  ┌─────────────────────────────────────────────────────┐   │
│  │              Creator Functions                       │   │
│  │  - Lambda functions for each effect type            │   │
│  │  - Call GetParams<T>() and make_shared              │   │
│  └─────────────────────────────────────────────────────┘   │
│  ┌─────────────────────────────────────────────────────┐   │
│  │              Fallback Mechanism                      │   │
│  │  - IsFallbackDisabled()                              │   │
│  │  - Original switch-case preserved                   │   │
│  └─────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────┘
```

## 三、核心接口

### 3.1 工厂类接口

```cpp
namespace OHOS {
namespace GraphicsEffectEngine {

class GEEffectFactory {
public:
    // 单例模式
    static GEEffectFactory& GetInstance();
    
    // 创建效果接口
    std::shared_ptr<Rosen::Drawing::IGEFilterType> CreateFilter(
        std::shared_ptr<Drawing::GEVisualEffectImpl> ve);
    
    std::shared_ptr<Rosen::Drawing::GEShaderMask> CreateMask(
        std::shared_ptr<Drawing::GEVisualEffectImpl> ve);
    
    std::shared_ptr<Rosen::Drawing::GEShaderShape> CreateShape(
        std::shared_ptr<Drawing::GEVisualEffectImpl> ve);
    
    // 回退控制
    static bool IsFallbackDisabled();
    static void SetFallbackDisabled(bool disabled);
    
private:
    GEEffectFactory() = default;
    
    // 注册表
    std::unordered_map<FilterType, FilterCreatorFunc> filterRegistry_;
    std::unordered_map<FilterType, MaskCreatorFunc> maskRegistry_;
    std::unordered_map<FilterType, ShapeCreatorFunc> shapeRegistry_;
};

} // namespace GraphicsEffectEngine
} // namespace OHOS
```

### 3.2 注册宏接口

```cpp
// 基础注册宏

// Shape注册宏
```

## 四、使用方法

### 4.1 基础效果注册

**最简单的注册方式（一行代码）：**

```cpp
// src/core/ge_effect_factory.cpp

GE_FACTORY_REGISTER(GEKawaseBlurShaderFilter)
GE_FACTORY_REGISTER(GEMESABlurShaderFilter)
GE_FACTORY_REGISTER(GEGreyShaderFilter)

// Mask效果
GE_FACTORY_REGISTER_MASK(GERippleShaderMask)
GE_FACTORY_REGISTER_MASK(GEDoubleRippleShaderMask)

// Shape效果
GE_FACTORY_REGISTER_SHAPE(GESDFRRectShaderShape)
GE_FACTORY_REGISTER_SHAPE(GESDFUnionOpShaderShape)
```

**原理：**
- 宏自动提取 `ClassName` 和 `ParamType`
- 通过 `GEFilterTypeInfo<ClassName>::ParamType` 获取参数类型
- 通过 `GEFilterParamsTypeInfo<ParamType>::ID` 获取枚举值
- 生成 Lambda 函数调用 `GetParams<T>()` 和 `make_shared`

### 4.2 自定义逻辑注册

**复杂效果需要自定义创建逻辑：**

```cpp
// KAWASE_BLUR: 根据系统属性切换到 MESA_BLUR
GE_FACTORY_REGISTER_CUSTOM(KAWASE_BLUR,
    [](GEEffectFactory::VisualEffectImplPtr ve) -> std::shared_ptr<Rosen::Drawing::IGEFilterType> {
        auto params = ve->template GetParams<Rosen::Drawing::GEKawaseBlurShaderFilterParams>();
        if (!GERender::IsMesablurAllEnabled()) {
            return std::make_shared<Rosen::GEKawaseBlurShaderFilter>(*params);
        }
        // 转换到 MESA_BLUR
        auto mesaParams = std::make_shared<Rosen::Drawing::GEMESABlurShaderFilterParams>();
        mesaParams->radius = params->radius;
        return std::make_shared<Rosen::GEMESABlurShaderFilter>(*mesaParams);
    })

// LINEAR_GRADIENT_BLUR: 条件外部加载
GE_FACTORY_REGISTER_CUSTOM(LINEAR_GRADIENT_BLUR,
    [](GEEffectFactory::VisualEffectImplPtr ve) {
        auto params = ve->template GetParams<Rosen::Drawing::GELinearGradientBlurShaderFilterParams>();
        if (params->isRadiusGradient) {
            // 尝试外部加载
            void* impl = GEExternalDynamicLoader::GetInstance().CreateGEXObjectByType(...);
            if (impl) return std::shared_ptr<IGEFilterType>(static_cast<IGEFilterType*>(impl));
        }
        return std::make_shared<Rosen::GELinearGradientBlurShaderFilter>(*params);
    })
```

### 4.3 外部效果注册

**支持动态加载外部效果库：**

```cpp
// 纯外部加载
GE_FACTORY_REGISTER_EXTERNAL(DOT_MATRIX, Rosen::Drawing::GEDotMatrixShaderParams)

// 外部加载 + 回退
GE_FACTORY_REGISTER_EXTERNAL_FALLBACK(MESA_BLUR,
    Rosen::Drawing::GEMESABlurShaderFilterParams,
    Rosen::GEMESABlurShaderFilter)
```

**外部加载流程：**
1. 通过 `GEExternalDynamicLoader` 加载外部库
2. 如果加载成功，返回外部效果对象
3. 如果加载失败（EXTERNAL_FALLBACK），返回内置效果

### 5.1 工厂调用集成

```cpp
const std::shared_ptr<Drawing::GEShaderMask> GEVisualEffect::GenerateShaderMask() const
{
    // 工厂优先创建
    auto factoryMask = GraphicsEffectEngine::GEEffectFactory::GetInstance().CreateMask(
        visualEffectImpl_);
    if (factoryMask) {
        GE_LOGI("[GEVisualEffect] GenerateShaderMask: factory created");
        return factoryMask;
    }
    
    // 回退机制
    if (GraphicsEffectEngine::GEEffectFactory::IsFallbackDisabled()) {
        GE_LOGE("[GEVisualEffect] GenerateShaderMask: factory failed, fallback disabled");
        return nullptr;
    }
    
    // 保留原有 switch-case 回退逻辑
    switch (impl->GetFilterType()) {
        case GEVisualEffectImpl::FilterType::RADIAL_GRADIENT_MASK:
            ...
    }
}

// GenerateShaderShape 类似
```

### 5.2 渲染管线流程（ge_render.cpp）

工厂已完整集成到渲染管线的核心函数：

#### GenerateShaderFilter

```cpp
std::shared_ptr<GEShaderFilter> GERender::GenerateShaderFilter(
    const std::shared_ptr<Drawing::GEVisualEffect>& vef)
{
    auto ve = vef->GetImpl();

    // 工厂优先策略：先尝试工厂
    shaderFilter = GEEffectFactory::GetInstance().CreateFilter(ve);
    if (shaderFilter) {
        LOGI("created via FACTORY");
        shaderFilter->SetShaderFilterCanvasinfo(vef->GetCanvasInfo());
        shaderFilter->SetSupportHeadroom(vef->GetSupportHeadroom());
        return shaderFilter;
    }

    // 回退禁用检查
    if (GEEffectFactory::IsFallbackDisabled()) {
        return nullptr;
    }

    // 回退旧通路（保留原有 switch-case）
    LOGE("using LEGACY path");
    switch (ve->GetFilterType()) { ... }
}
```

#### GenerateShaderEffect

```cpp
std::shared_ptr<GEShader> GERender::GenerateShaderEffect(
    const std::shared_ptr<Drawing::GEVisualEffectImpl>& ve)
{
    // 工厂优先策略
    auto shader = GEEffectFactory::GetInstance().CreateShader(ve);
    if (shader) {
        LOGI("created via FACTORY");
        return shader;
    }

    // 回退禁用检查
    if (GEEffectFactory::IsFallbackDisabled()) {
        return nullptr;
    }

    // 回退旧通路（保留原有 shaderCreatorLUT）
    LOGE("using LEGACY path");
    auto it = shaderCreatorLUT.find(ve->GetFilterType());
    return it != shaderCreatorLUT.end() ? it->second(ve) : nullptr;
}
```

## 七、性能分析

### 7.1 性能对比

| 方式 | 查找复杂度 | 添加新效果 | 维护成本 |
|------|-----------|----------|----------|
| **Switch-case** | O(n) - 线性查找 | 修改多处代码 | 高 - 分散维护 |
| **工厂模式** | O(1) - 哈希查找 | 添加一行注册 | 低 - 集中维护 |

### 7.2 性能优势

1. **查找性能**：
   - Switch-case: 线性查找 O(n)，随着效果数量增加性能下降
   - 工厂模式: 哈希查找 O(1)，性能稳定不受效果数量影响

2. **编译优化**：
   - 哈希表在编译期生成，运行时直接使用
   - 无动态内存分配开销

3. **内存占用**：
   - 注册表大小固定（基于效果类型数量）
   - 每个 Lambda 函数编译期优化

## 八、扩展机制

### 8.1 外部动态加载

**支持动态加载外部效果库：**

```cpp
// 外部效果加载流程
std::shared_ptr<IGEFilterType> CreateExternalEffect(FilterType type, const void* params) {
    // 1. 通过动态加载器加载
    void* impl = GEExternalDynamicLoader::GetInstance().CreateGEXObjectByType(
        static_cast<uint32_t>(type), 
        sizeof(ParamType), 
        params);
    
    // 2. 转换为智能指针
    if (impl) {
        return std::shared_ptr<IGEFilterType>(static_cast<IGEFilterType*>(impl));
    }
    
    // 3. 失败时回退
    return nullptr;
}
```

### 8.2 外部效果库要求

**外部效果库需要实现：**
- 导出 `CreateGEXObjectByType` 函数
- 支持参数序列化/反序列化
- 符合效果接口规范


3. **外部效果使用外部宏**
   ```cpp

   ```

3. **添加工厂注册**
   ```cpp
   // src/core/ge_effect_factory.cpp 中添加
   ```

