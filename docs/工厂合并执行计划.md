## 四、关键设计决策

1. **工厂完全接管** - 无回退逻辑，工厂创建失败直接返回nullptr
2. **集中注册** - 所有效果注册集中在 `ge_effect_register.cpp`
3. **宏注册** - 使用 `GE_FACTORY_REGISTER` 系列宏简化注册
4. **分类组织** - 按字母序排列头文件，按分类分组注册

## 五、关键接口

工厂宏依赖的接口全部存在：
- `DECLARE_GEFILTER_TYPEFUNC` - 效果类中声明
- `GEFilterTypeInfo<T>::ParamType` - 参数类型萃取
- `GEFilterParamsTypeInfo<ParamType>::ID` - 参数类型ID
- `GetParams<T>()` - 参数获取方法
